# Prompt Generate: Detailed Deck — RAKERWIL PW Jama'ah Al Khidmah DKI Jakarta

## Instruksi untuk NotebookLM

Buatlah slide deck presentasi dengan gaya **detailed** (lengkap, informatif, data-rich) untuk **Rapat Kerja Wilayah (RAKERWIL) PW Jama'ah Al Khidmah DKI Jakarta**.

### Format & Gaya
- **Bahasa**: Bahasa Indonesia, formal dengan nuansa keagamaan
- **Tone**: Profesional, transparan, informatif, mengajak
- **Jumlah slide**: 18–20 slide
- **Gaya detailed**: tampilkan data lengkap, tabel, detail nomor — ini untuk dokumen kerja yang bisa dibaca sendiri
- **Target audiens**: Pengurus PD/PC, jama'ah, dan pihak yang perlu detail lengkap

### Sumber yang Harus Dibaca
Prioritaskan file berikut (upload semua ke NotebookLM):
1. `brief.md` — Identitas acara, speaker, audiens, tujuan
2. `structure.md` — Arc narasi dan sekuens section
3. `detailed.md` — Outline slide-by-slide (detail version)
4. Semua file `.md` di `references/` — Data keuangan, majlis, strategi, sarung, profil organisasi
5. File gambar di `references/` (`poster-masjid-al-fithrah.jpeg`, `peta-jabodetabek.png`, `chart-cashflow-koperasi.jpeg`, `sarung-motif-1.jpeg`, `sarung-motif-2.jpeg`, `sarung-motif-3.jpeg`)
6. ⚠️ File `portrait-kh-achmad-asrori.png` TIDAK perlu diupload ke NotebookLM (gambar wajah) — akan ditambahkan manual di PowerPoint

### Aturan Konten
1. Tampilkan **seluruh data keuangan** dalam tabel (tidak diringkas)
2. Sertakan jadwal detail untuk setiap lokasi majlis. **JANGAN mengarang alamat** — gunakan hanya data yang tersedia di sumber
3. Kutipan Hadrotusyeikh: tampilkan lengkap dengan sumber. JANGAN sertakan foto wajah — akan ditambahkan manual
4. Detail harga sarung, margin, dan skema pembagian profit — transparent dan jelas. HPP Rp 65.000 sudah termasuk royalti Rp 2.000
5. Progress tanah: tampilkan angka-angka lengkap (luas, harga/m², total, dibayar, sisa, cicilan)
6. Setiap slide boleh berisi lebih banyak teks, tapi tetap terstruktur rapi
7. Sertakan catatan/keterangan penting dari data sumber
8. Gunakan bahasa yang hangat dan transparan
9. **JANGAN mengarang data yang tidak ada di sumber** — jika informasi tidak tersedia, tuliskan bahwa data belum tersedia
10. Arc narasi: **"Dari Menjaga ke Menyelesaikan — Dari Istiqomah ke Inovasi"** harus tercermin dalam alur slide
11. Frame sarung sebagai **ikhtiar menopang keberlanjutan syiar**, bukan sekadar usaha komersial
