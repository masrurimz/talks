# Prompt Revisi Deck NotebookLM

Catatan penting: menurut bantuan resmi NotebookLM, revisi slide saat ini **tidak lagi mempertimbangkan sources**. Jadi, bila arah deck sudah melenceng jauh, lebih aman **generate deck baru** dengan prompt yang diperbaiki daripada mengandalkan revisi berat.

## Prompt revisi 1 — kembalikan fokus ke media sosial
```text
Kembalikan fokus deck ini ke media sosial sebagai isu utama. Kurangi porsi game online dan judi online agar tetap hanya menjadi bahasan singkat pendukung.
```

## Prompt revisi 2 — sesuaikan untuk sesi 1 acara dua pemateri
```text
Ubah deck ini agar jelas terasa sebagai sesi 1 yang membuka fenomena dari sudut pandang praktisi / akademisi bidang Geomatika. Jangan membuat deck terasa seperti sudah menuntaskan kajian agama. Penutup harus menjembatani ke sesi 2.
```

## Prompt revisi 3 — buat lebih lengkap, jangan terlalu pendek
```text
Perbanyak cakupan deck ini agar tidak terlalu pendek. Lebih baik sedikit lebih banyak slide daripada terlalu ringkas, selama tetap fokus dan mudah dipahami.
```

## Prompt revisi 4 — tambahkan penjelasan teknis yang sederhana
```text
Tambahkan sedikit penjelasan teknis yang mudah dipahami tentang notifikasi, feed tanpa akhir, algoritma rekomendasi sederhana, personalisasi konten, autoplay, FOMO, dan social proof.
```

## Prompt revisi 5 — sederhanakan bahasa untuk audiens umum desa
```text
Sederhanakan seluruh deck untuk audiens umum di lingkungan desa Indonesia. Kurangi istilah teknis, pendekkan kalimat, dan gunakan contoh yang dekat dengan kehidupan sehari-hari.
```

## Prompt revisi 6 — kuatkan bagian pembatasan usia dan pengawasan
```text
Perjelas alasan mengapa pembatasan usia dan pengawasan pengguna muda itu penting. Mulai dari alasan perlindungan dulu, lalu beri contoh singkat praktik umum atau platform tanpa menjadikannya fokus utama.
```

## Prompt revisi 7 — perkuat jembatan ke sesi 2
```text
Buat slide penutup lebih reflektif. Tegaskan bahwa problem digital ini menunjukkan manusia membutuhkan arah, lalu arahkan audiens ke sesi berikutnya tentang bagaimana Al-Qur'an menjawab problematika zaman.
```

## Jika deck awal kurang tepat
Gunakan prompt generasi baru seperti ini:

```text
Buat ulang deck dari awal untuk sesi 1 acara refleksi Nuzulul Qur'an bertema “Al-Qur'an menjawab problematika zaman”. Fokus utama harus pada media sosial, dengan sedikit bahasan tentang game online, judi online, pentingnya pengawasan pengguna muda, dan sedikit penjelasan teknis yang mudah dipahami. Buat deck yang cukup lengkap untuk 30 menit, lalu akhiri dengan jembatan yang natural ke sesi 2.
```
